Boogie Worms

I worked on this game as my summer project of 2018. It consists of a combination of the old code I learned in my first year at RIT and new code I taught myuself with the help of online refrences. The game itself is focused on using the cursor to catch the "boogie worms" on the screen. Every five rounds, an element of the game changes to increase the difficulty, such as mouse size, worm speed, worm location, etc.

There are some glitches in the game I hope to fix. I also hope to add new levels, as well as include a timer visable on the screen.
