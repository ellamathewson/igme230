Sea What You Did?

This game was created as a final project for my Website Design and Implementation class. The purpose of the project is to educate people about ocean pollution with an interactive experience. Our intended audience is focused primarily on a younger age group, but the program can be used by anyone.

We used HTML, CSS, and two JavaScript libraries -- Particle.js and Hexi.js -- to create this program. A more descriptive breakdown of how we divided the work within our team is on the documentation page of the project.

One error in this program is that the program itself is not centered on the screen. Because Hexi.js is an outdated library, we had difficulty fixing this issue, and due to time limitations, left it as is. Along with this, the program does not run on Firefox. However, it does run on Chrome and Safari well. In the future we hope to fix these issues if possible.
