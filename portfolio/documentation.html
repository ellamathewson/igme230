    I used this source for a specific box-shadow line of CSS, which is used for the nav bar across every page. I liked the shadow effect it had.

Challenges:

Project Nav Bar

    This aspect to my website was particularly challenging. The nav bar on the projects page has the ability to sort the projects based on what the project is. I divided it into four main groups: JavaScript, HTML and CSS, Design, and C#. I ended up removing the C# category because as of right now I do not have any C# projects completed.
    Using JavaScript, when the link is pressed, I needed certain projects to disappear from the page and certain ones to stay. This was challenging because I needed to get the id of every project and look for specific ones. I also needed to know how to obtain the correct number of projects that were on the page.

Responsiveness

    This aspect was challenging in a different way. It was challenging to try and rework all of my pages for a different window size, as a lot of aspects needed to be changed in order for the content to look right.
